História do Bootstrap

Bootstrap é um framework muito utilizado no front-end, este disponibiliza muitas funcionalidades prontas, como: modais, formulários, menus de navegação, além de incluir responsividade em seus elementos. O framework faz uso de JavaScript e CSS para implementar os mecanismos e estilizações, proporcionando rapidez, proatividade e inovação ao desenvolver páginas web. Vale ressaltar que o Bootstrap é uma ferramenta gratuita, logo, seu código-fonte é aberto, bem como, pode ser utilizado em aplicações desenvolvidas com outras ferramentas e linguagens, como: .Net, Ruby, PHP e Angular.

Os engenheiros Jacob Thorton e Mark Otto foram os responsáveis pela criação do Bootstrap, em 2010. Surgiu para tentar solucionar uma incompatibilidade interna da própria equipe. De início, havia o objetivo de otimizar o desenvolvimento utilizando uma estrutura única, reduzindo as inconsistências entre os códigos dos integrantes da equipe. Com o sucesso, lançaram-a no GitHub, em 2011, como um software livre, que se tornou muito popular, a ponto de ser uma das ferramentas mais populares da plataforma no começo do ano seguinte, 2012.

Bootstrap é um framework de muita importância para a elaboração de websites. É considerado uma referência devido suas tendências de design, princípios de usabilidade e padrões para plataformas, além da possibilidade de reutilização de código, proporcionando economia de tempo. Vale ressaltar que o framework permite o desenvolvimento de websites harmônicos e impressionantes.

Tecnologias

Bootstrap.js

Para que funcionem de maneira correta, certos componentes e recursos do framework precisam executar algumas funções em JavaScript. Como exemplos, temos: modais, que mostram mensagens de alerta e botões ao usuário; o controle de paginação e diversas outras funcionalidades do Bootstrap. Para que isso seja possível, ele contém o arquivo denominado "bootstrap.js".

Bootstrap.css

Esse é o principal arquivo de estilo da ferramenta. Nele estão contidas as classes e variáveis pré-definidas do framework, que servem para criar os layouts e formatação dos componentes das páginas ja incluindo a responsividade. Para que seu site fique ainda mais personalizado você também pode adicionar suas linhas de código CSS, mas recomendamos que o faça em um arquivo separado para que não haja uma interferência drástica na ferramenta que acarrete o seu mau funcionamento.

Glyphicons

Ícones são fundamentais para deixar um website mais agradável e atraente, além de deixar a plataforma visivelmente mais elegante e moderna. Para isso, o Bootstrap usa o Glyphicons, que é uma biblioteca a ser utilizada em conjunto com o framework, proporcionando uma série de elementos gratuitos. A biblioteca também conta com pacotes de ícones pagos, permitindo uma gama de possibilidades ainda maior.

Conhecimentos básicos

Ao se tratar de Bootstrap, faz-se uso das linguagens HTML, CSS e JavaScript. Portanto, seria interessante ter um conhecimento básicos destas, de modo a facilitar o desenvolvimento e utilização do framework. Também seria vantajoso estudar sobre o próprio Bootstrap, ler a documentação, e ver seus componentes e funcionalidades, para se ter uma maior clareza e agilidade ao utilizá-lo.

Ambiente de desenvolvimento

Antes de fazer o download e a implementação do Bootstrap para seu website, é necessário verificar se seu ambiente de desenvolvimento é adequado. Não é necessário uma IDE super elaborada ou até mesmo pesada, apenas um local para elaboração dos códigos HTML, CSS e JavaScript. Recomendamos os seguintes ambientes: Visual Studio Code, Sublime Text, Notepad++ e Atom.
